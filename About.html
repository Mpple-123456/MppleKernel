🚀 快速开始

1️⃣ 安装依赖

sudo ./installbash.sh

自动安装 g++, nasm, qemu, python3-tk, grub, xorriso 等。

2️⃣ 构建内核

cd build
./build.sh --grub -a

或使用图形化工具：python3 make.py

3️⃣ 运行测试

./build.sh -r

QEMU 将自动启动，优先使用 ISO 镜像。

4️⃣ 写入 USB

sudo ./install.sh

⚠️ 会覆盖设备所有数据，谨慎操作。

📟 基础命令

启动后提示符为 >，支持以下命令：

命令说明
help显示帮助信息
clear清屏
echo <text>回显文本
ver显示内核版本
poweroff / shutdown关机
reboot重启
RESETKB重置键盘状态
desktop进入桌面环境

使用技巧：退格删除、Caps Lock 切换大写（屏幕右上角显示 CAPS）、Shift 输入符号。

🧠 内核架构摘要

  • 引导程序加载内核到 0x10000，进入保护模式。
  • 直接操作 VGA 显存 0xB8000，硬件光标同步。
  • 键盘轮询，扫描码映射 ASCII，支持 Shift/Caps。
  • 错误处理：panic() 显示红色信息并重启。
  • Multiboot 头部（可选），支持 GRUB 引导。
